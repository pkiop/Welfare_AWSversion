# BOKJIDA
사용자 맞춤 복지 정보 추천 서비스 제공 앱


## USAGE

1. 회원가입시 입력한 정보를 토대로 받을 수 있는 복지 정보 추천
2. 챗봇을 활용한 복지 정보 검색
3. 복지 정보 신청 알람 서비스
